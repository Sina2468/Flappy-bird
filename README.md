## Installation

1. Download the program 

        git clone https://github.com/Sina2468/Flappy-bird.git

* or

    https://github.com/Sina2468/Flappy-bird/archive/refs/heads/main.zip

2. Run **Setup.exe** to install it


## Usage

* Run **Flappy bird.exe** 

    * Press **R** for Start
    * Press **Space** for jump
